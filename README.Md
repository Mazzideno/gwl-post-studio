## ✨ Features

### 🎯 Core Features

- **🌓 Dark/Light Mode Toggle** - Switch between themes seamlessly
- **🎨 Drag & Drop Canvas Editor** - CorelDRAW-style interface
- **📐 Shape Tools** - Rectangle, Circle, Triangle, Line, Star
- **🖼️ Logo Upload** - Add and position your company logo
- **📝 Text Formatting** - Uppercase, lowercase, title case for each field
- **↔️ Text Alignment** - Left, center, right alignment options
- **🎭 Template Styles** - 6 professional templates:
  - Reality Check
  - Bold Quote
  - Myth vs Truth
  - Language Fact
  - Culture Drop
  - Announce

### 🎨 Design Features

- **Color Customization** - Shape colors, backgrounds, accents
- **Background Styles**:
  - Solid Color
  - Gradient
  - Geometric Pattern
  - Radial Gradient
  - Diagonal Stripes
- **Canvas Sizes**:
  - Square (1080×1080) - Instagram
  - Portrait (1080×1350) - Instagram Portrait
  - Story (1080×1920) - Instagram/FB Stories
  - Landscape (1200×630) - Facebook/Twitter

### 💾 Export Options

- Download as PNG
- Download as JPG
- Copy caption to clipboard
- Auto-save to local storage
